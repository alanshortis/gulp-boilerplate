# Gulp Boilerplate

Boilerplate for gulp tasks. In progress.

### Tasks

`css`
* SASS compilation.
* Soucemaps.
* Autoprefix CSS.

`minify`
* Minify CSS into a new file.

`image`
* Optimize images.

`watch`
* Watch SASS file for changes and run `css`.

`clean`
* Remove the entire `dist` folder.

`default`
* Run all tasks (aside from `watch`).
